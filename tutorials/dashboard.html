Master the SPIDEY BOT admin dashboard to manage your server settings, monitor statistics, and configure features.

1
Login to Dashboard

Visit /login and click "Login with Discord".

Authorize the bot to access your account information.

üí° You must be an admin in at least one server to access the dashboard

2
Select Your Server

After login, you'll see a server dropdown at the top of the dashboard.

Click it to see all servers where you're an admin.

Select your server - the dashboard will load settings for that server only.

üí° You can switch servers anytime using the dropdown

3
Dashboard Home - View Statistics

The Dashboard tab shows live server statistics:

  • Member Count: Total members in your server
  • Verified Members: Members with the @Members role
  • Bots: Bot accounts in the server
  • Admins & Mods: Staff member counts
  • Activity Graph: Visual member activity

4
Navigate Sidebar Sections

The left sidebar contains all configuration categories:

  • Settings: Basic config (prefix, language, timezone)
  • Logging: Set up moderation & activity logs
  • Server Guard: Anti-spam & raid protection
  • React Roles: Reaction-based role assignment
  • Server Messages: Welcome/goodbye messages
  • XP Levels: Leveling system configuration
  • ...and more! 18 total configuration sections

5
Edit a Configuration

Click any section in the sidebar (e.g., "Settings").

Fill in the form fields with your desired values.

Notice: An "Unsaved Changes" notification appears at the bottom.

üí° Changes are NOT saved until you click "Save Changes"

6
Save Your Changes

When you see the unsaved notification at the bottom, click "Save Changes".

The notification will disappear once saved.

  • "Reset" button: Reverts unsaved changes only
  • "Save Changes" button: Sends to bot & Discord server

7
Verify Changes Live

Your changes take effect immediately on Discord!

Example: Change the prefix in Settings ‚Üí Save ‚Üí Use new prefix in Discord.

‚úÖ Most settings apply instantly to your server

8
UI Customization (Optional)

Visit the "UI Colors" tab to customize the dashboard.

  • Choose from 25+ color themes
  • Adjust accent colors
  • Change text colors
  • Toggle light/dark mode

All customizations are saved to your browser.

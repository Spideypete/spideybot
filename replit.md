# Spidey Bot - Discord Bot

## Overview
A Discord bot built with Node.js and discord.js. The bot responds to commands like `!ping` with "Pong!".

## Project Architecture
- **Language**: Node.js (CommonJS)
- **Main Dependencies**: 
  - discord.js v14.25.0 - Discord API wrapper
  - dotenv v17.2.3 - Environment variable management
- **Entry Point**: index.cjs

## Setup Requirements
- Requires a Discord bot TOKEN (stored as environment variable)
- The bot needs Message Content Intent enabled in Discord Developer Portal

üìä How to Create a Dashboard Preview

Learn how to create a professional preview image showcasing your admin dashboard features.

1
Prepare Your Dashboard

Set up your dashboard to look its best:

  • Log into the admin dashboard with a populated server
  • Make sure you have real data (members, activity, etc.)
  • Navigate to a visually interesting section like:
    • Dashboard home with graphs and stats
    • Analytics or member leaderboard
    • A configuration panel with color customization
üí° Real data makes the dashboard look more impressive and useful

2
Optimize the View

Prepare your screen for a professional screenshot:

  • Close all browser tabs except the dashboard
  • Hide bookmarks bar and address bar if possible
  • Scroll to show the most important section
  • Set zoom to 100% (Ctrl/Cmd + 0)
  • Make sure colors are vibrant and clear
üí° A clean interface shows professionalism and attention to detail

3
Select the Best Section

Choose which part of the dashboard to showcase:

  • Home Page: Shows stats, graphs, live data
  • Settings Panel: Shows customization options
  • Analytics: Shows member activity and trends
  • Configuration Forms: Shows how easy it is to manage

Best choice: Home Page with stats - it shows the most impressive data

4
Take a Full Screenshot

Capture your dashboard:

  • Windows: Snipping Tool or Win+Shift+S
  • Mac: Cmd+Shift+4 or Screenshot app
  • Linux: Flameshot or Gnome Screenshot

Make sure to capture:

  • The header with server name and user info
  • Sidebar with section menu
  • Main content area with stats/graphs
  • Any colorful elements or charts

5
Crop to Card Size

Edit your screenshot to fit the preview card dimensions:

  • Target size: 600x400px (landscape)
  • Focus on the central dashboard content
  • Crop out empty margins and whitespace
  • Keep all the important UI elements visible

Use an editor like:

  • Photoshop or GIMP (professional)
  • Canva (online, user-friendly)
  • Paint.NET (simple and fast)

6
Enhance Visual Appeal

Make your preview stand out:

  • Adjust brightness: Make cyan/purple colors more vibrant
  • Increase saturation: Colors pop more on the website
  • Add subtle glow: Optional purple border effect
  • Add overlay text: "Admin Dashboard" label in corner
üí° Visual enhancements help users immediately understand what they're looking at

7
Optimize File Size

Compress the image for web use:

Recommended tools:
- TinyPNG.com
- Compressor.io
- ImageOptim (Mac)

Target: Keep under 500KB for fast loading times

  • Quality should remain sharp and clear
  • Colors should not become washed out

8
Save & Upload to Site

Add the preview to your website:

/public/assets/dashboard-preview.png

Update your HTML card:

<img src="/assets/dashboard-preview.png" alt="Admin Dashboard">

Add styling to ensure proper sizing:

style="width: 100%; height: 180px; object-fit: cover; border-radius: 8px;"

9
Test & Verify

Check your preview on the actual website:

  • View on desktop (looks good at full size)
  • View on mobile (maintains aspect ratio)
  • Check load time (should be instant)
  • Verify hover effects (card lifts up)

If anything looks off, go back to step 5 and refine the crop.
